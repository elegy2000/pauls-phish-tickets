# Ticket Stub Image Guide

## How to Name Your Ticket Stub Images

1. When you scan a ticket stub, name it using this simple format:
   - Example: If the show was on October 30, 1983, name the file: `october-30-1983.jpg`
   - Another example: For December 2, 1983, name it: `december-2-1983.jpg`

2. Important rules for naming:
   - Use lowercase letters
   - Use hyphens (-) between words
   - Don't use any commas
   - Always include the full date (month-day-year)
   - Always save as .jpg files

## How to Save Your Images

1. Save all your scanned ticket stubs in this folder (the folder where you found this guide)
2. Make sure each image is clear and well-lit
3. Try to keep the file size reasonable (under 1MB per image is good)

## Common Examples

Here are some examples of how to name your files:
- For a show on July 4, 1999: `july-4-1999.jpg`
- For a show on December 31, 1999: `december-31-1999.jpg`
- For a show on March 15, 2000: `march-15-2000.jpg`

## Need Help?

If you're not sure about the exact date of a show, you can check the CSV file that was given to you. The date column will show you exactly how to name the file.

Remember: The website will automatically find and display your ticket stub images as long as you follow this naming format!
